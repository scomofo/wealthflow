AI-assisted development. They captured the design and implementation plan
for each feature/phase as it was built.

They're kept for reference but are **not** authoritative — the code, tests,
and `CLAUDE.md` are the source of truth for current behavior. Moved here from
the repo root to keep the top level clean.
